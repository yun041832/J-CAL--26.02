        달력·루틴·메모·위젯 사용 방법과 활용 팁을 정리했습니다.

        웹 달력으로 월별 일정 관리하는 방법

        월별 그리드 확인

        Jay Calendar 메인 화면에서 달력 메뉴를 선택하면 현재 월의 날짜 그리드가 표시됩니다. 각 칸에는 해당 날짜에 등록된 일정·할 일이 요약되어 보이므로, 한 달 단위로 계획을 파악할 수 있습니다.

        날짜 선택과 일정 등록

        원하는 날짜 칸을 클릭하면 오른쪽 또는 하단에 일정 등록 패널이 나타납니다. 일정 탭과 할 일 탭을 구분해 사용할 수 있으며, 제목 입력 후 Enter로 추가하면 해당 날짜에 반영됩니다.

        다른 월로 이동

        상단의 이전 월·다음 월 버튼으로 다른 달을 선택할 수 있습니다. 월이 바뀌어도 등록된 데이터는 그대로 유지되며, 필요 시 시작일·종료일을 설정해 여러 날에 걸친 일정을 넣을 수 있습니다.

        이달의 목표를 달력 상단에 두는 활용법

        목표 입력 위치

        달력 화면 상단에는 이달의 목표를 입력하는 칸이 있습니다. 여기에 한 줄 목표 문구를 입력하고 Enter를 누르면 저장되며, 달력을 볼 때마다 상단에서 확인할 수 있습니다.

        활용 상황

        월 초에 이번 달에 꼭 지킬 원칙이나 목표를 적어 두고, 매일 달력을 열 때 상단 문구를 보며 동기 부여를 하는 용도로 쓸 수 있습니다. 일정만 보는 것이 아니라 목표를 함께 상기시키는 역할을 합니다.

        수정 및 삭제

        목표 문구 옆 메뉴 버튼을 누르면 스타일 변경이나 내용 수정이 가능합니다. 삭제 시 해당 칸을 비우면 되며, 데이터는 브라우저 저장소에 저장되어 새로고침 후에도 유지됩니다.

        할 일과 일정을 구분해 등록하는 방법

        일정 탭과 할 일 탭

        날짜를 선택했을 때 보이는 패널에는 일정 탭과 할 일 탭이 있습니다. 일정은 날짜·시간이 있는 이벤트 성격의 항목에, 할 일은 체크 가능한 작업 목록에 적합합니다.

        일정 등록 시 유의점

        일정은 배경색을 지정할 수 있어 달력 그리드에서 색으로 구분해 볼 수 있습니다. 시작일·종료일을 다르게 넣으면 여러 날에 걸쳐 한 줄로 표시됩니다.

        할 일 체크

        할 일은 완료 시 체크하면 취소선 등으로 표시되어 진행 상황을 한눈에 볼 수 있습니다. 목록에서 삭제하지 않고 체크만 해 두면 나중에 무엇을 했는지도 확인할 수 있습니다.

        루틴을 주간 단위로 설정하고 체크하는 방법

        루틴 메뉴 진입

        사이드바에서 루틴 메뉴를 클릭하면 주간 달력과 루틴 목록이 표시됩니다. 매일 반복 또는 특정 요일만 반복하는 루틴을 추가할 수 있습니다.

        루틴 항목 추가

        추가 버튼을 누른 뒤 루틴 이름, 반복 요일(매일/특정 요일), 색상·이모티콘을 설정합니다. 저장하면 주간 그리드에 해당 요일마다 표시되어 체크할 수 있습니다.

        주간 뷰에서 체크

        각 요일 칸에서 해당 날짜에 수행한 루틴을 체크하면 완료로 기록됩니다. 한 주 단위로 채워 나가면 습관 이행률을 시각적으로 확인할 수 있습니다.

        메모를 날짜별로 정리하는 방법

        메모 메뉴와 날짜 선택

        메모 메뉴를 선택하면 날짜별 메모 목록을 볼 수 있습니다. 날짜를 선택하거나 오늘 날짜 기준으로 해당 날짜에 저장된 메모가 목록으로 나옵니다.

        새 메모 작성

        새 메모 추가 버튼을 누르면 제목과 본문을 입력하는 화면이 나옵니다. 제목 옆에서 색상·이모티콘을 지정할 수 있으며, 저장 시 선택한 날짜에 연결됩니다.

        목록에서 확인

        메모 목록에서는 제목과 요약이 보이므로, 원하는 메모를 골라 클릭해 내용을 보거나 위젯으로 띄울 수 있습니다. 날짜를 바꿔가며 과거·미래 날짜의 메모도 조회할 수 있습니다.

        위젯으로 달력만 띄워 쓰는 방법

        위젯 버튼 위치

        달력 화면에서 상단의 위젯(팝업) 버튼을 누르면 작은 달력 창이 새 창 또는 레이어로 열립니다. 메인 화면은 그대로 두고 달력만 따로 띄워 두고 싶을 때 사용합니다.

        사용 상황

        다른 작업을 하면서도 달력을 옆에 두고 날짜를 참고하거나, 선택한 날짜의 일정을 빠르게 확인할 때 유용합니다. 창 크기는 브라우저에 따라 조절할 수 있습니다.

        닫기 및 동기화

        위젯 창을 닫아도 메인 화면의 데이터에는 영향이 없습니다. 같은 브라우저·같은 저장소를 쓰므로 메인에서 수정한 내용은 위젯을 다시 열면 반영됩니다.

        여러 타이머를 동시에 쓰는 활용

        타이머 페이지 구성

        타이머 메뉴를 선택하면 여러 개의 타이머 블록이 그리드로 배치됩니다. 각 블록은 독립적으로 시·분·초를 설정하고 시작·일시정지·리셋할 수 있습니다.

        동시 사용 예시

        공부 25분, 휴식 5분, 운동 30분처럼 서로 다른 구간을 각각 다른 타이머에 설정해 두고 번갈아 켜서 사용할 수 있습니다. 한 타이머가 동작하는 동안 다른 타이머는 대기 상태로 두면 됩니다.

        위젯으로 띄우기

        각 타이머 블록의 위젯 버튼을 누르면 해당 타이머만 작은 창으로 띄울 수 있어, 화면 한쪽에 두고 다른 작업을 하며 시간을 재는 데 쓸 수 있습니다.

        달력에서 날짜 클릭 후 일정 추가 절차

        날짜 클릭

        달력 그리드에서 원하는 날짜 칸을 클릭하면 해당 날짜가 선택되고, 오른쪽 또는 하단에 일정·할 일 입력 영역이 나타납니다.

        입력 필드 사용

        할 일 입력란에 내용을 적고 Enter를 누르거나 추가 버튼을 누르면 목록에 들어갑니다. 일정 탭에서는 시작·종료 날짜와 시간, 색상 등을 선택할 수 있습니다.

        저장 방식

        입력한 내용은 자동으로 브라우저 저장소에 저장됩니다. 별도 저장 버튼을 누를 필요 없으며, 새로고침 후에도 해당 날짜에 등록된 항목이 유지됩니다.

        반복 루틴 설정 시 유의할 점

        반복 주기 선택

        루틴 추가 시 매일 반복인지, 특정 요일만 반복인지 선택합니다. 요일을 여러 개 고르면 해당 요일마다 주간 그리드에 표시되어 체크할 수 있습니다.

        이름과 구분

        루틴 이름은 짧게 구분 가능하게 적는 것이 좋습니다. 색상과 이모티콘으로 시각적 구분이 되므로, 아침 루틴·저녁 루틴처럼 성격별로 나누어 등록하면 확인하기 쉽습니다.

        순서 변경

        루틴 목록의 순서는 드래그 등으로 변경할 수 있는 경우가 있습니다. 자주 보는 루틴을 위쪽에 두면 주간 뷰에서 확인할 때 편리합니다.

        메모 목록에서 원하는 메모만 위젯으로 여는 방법

        메모 목록 확인

        메모 메뉴에서 날짜를 선택하면 그날짜에 저장된 메모 목록이 나옵니다. 각 메모 카드에는 제목과 일부 내용이 보이며, 위젯 버튼이 있습니다.

        위젯 버튼 클릭

        해당 메모의 위젯(팝업) 버튼을 누르면 그 메모만 새 창으로 열립니다. 메인 화면은 그대로 두고 메모 내용만 따로 보고 싶을 때 사용합니다.

        여러 메모 동시에 띄우기

        필요하면 여러 메모를 각각 위젯으로 띄워 두고 작업할 수 있습니다. 각 창은 독립적이며, 메인 화면에서 메모를 수정한 뒤 위젯을 다시 열면 변경 내용이 반영됩니다.

        브라우저 저장소에 데이터가 저장되는 방식

        로컬 저장

        Jay Calendar는 서버 로그인 없이 브라우저의 로컬 저장소(localStorage 등)를 사용합니다. 일정·할 일·루틴·메모는 이용자가 사용하는 기기·브라우저에만 저장됩니다.

        동기화 범위

        같은 브라우저에서 여러 탭을 열면 저장소를 공유하므로 탭 간에 최신 데이터가 반영될 수 있습니다. 다른 기기나 다른 브라우저에서는 별도로 데이터가 쌓이므로 동기화되지 않습니다.

        백업과 삭제

        데이터를 남기려면 브라우저 데이터를 삭제하지 않도록 주의합니다. 기기를 바꾸거나 브라우저를 초기화하면 해당 환경에 저장된 데이터는 복구할 수 없습니다.

        일정에 색상과 이모지를 넣는 방법

        색상 선택

        일정·할 일을 등록할 때 색상 선택 버튼을 누르면 배경색을 고를 수 있습니다. 달력 그리드와 목록에서 해당 색으로 표시되어 구분이 쉽습니다.

        이모지 선택

        이모지 버튼을 누르면 자주 쓰는 이모지 목록이 나옵니다. 항목 앞에 이모지를 붙이면 목록에서 한눈에 종류를 구분할 수 있습니다.

        활용 예

        업무는 파란색, 개인 일정은 초록색처럼 카테고리별로 색을 통일하거나, 중요한 할 일에만 이모지를 붙여 강조하는 식으로 활용할 수 있습니다.

        주간 루틴 체크리스트 확인 순서

        주간 그리드 보기

        루틴 메뉴에서는 한 주 단위로 요일이 나열되고, 각 요일 칸에 그날 수행할 루틴 항목이 표시됩니다. 요일 순서대로 왼쪽에서 오른쪽으로 확인하면 됩니다.

        체크 순서

        아침에 루틴 메뉴를 열어 오늘 날짜 칸의 항목을 하나씩 체크해 나가면 됩니다. 순서는 사용자가 정한 루틴 목록 순서를 따르며, 필요하면 목록 순서를 바꿀 수 있습니다.

        이전 주 확인

        주간 뷰에서 이전 주·다음 주로 이동하면 과거 주의 체크 현황도 볼 수 있어, 주별 이행률을 점검할 수 있습니다.

        달력 그리드에서 오늘 날짜 찾기

        오늘 버튼

        달력 상단에 오늘 버튼이 있습니다. 클릭하면 현재 월로 이동하고 오늘 날짜가 선택된 상태로 표시되며, 해당 날짜의 일정·할 일 패널이 열립니다.

        다른 달에 있을 때

        다른 월을 보고 있을 때도 오늘 버튼을 누르면 오늘이 포함된 월로 돌아갑니다. 오늘이 해당 월에 없으면 자동으로 현재 월로 이동합니다.

        시각적 구분

        일부 테마에서는 오늘 날짜 칸에 테두리나 배경이 다르게 표시되어 그리드 안에서 빠르게 찾을 수 있습니다.

        할 일 기한(시작·종료일) 설정 방법

        상세 설정 열기

        할 일을 등록할 때 입력란 옆의 상세 설정(옵션) 버튼을 누르면 시작일·종료일 입력 필드가 나타납니다.

        시작일과 종료일

        시작일만 넣으면 그날에만 표시되고, 종료일까지 넣으면 시작일부터 종료일까지 달력에 연속으로 표시됩니다. 기한이 있는 과제나 이벤트에 유용합니다.

        표시 위치

        기간이 설정된 할 일은 달력 그리드에서 해당 날짜 칸들에 모두 표시되므로, 며칠에 걸친 작업을 한눈에 확인할 수 있습니다.

        메모 제목과 내용 작성 요령

        제목의 역할

        메모 제목은 목록에서 구분용으로 쓰입니다. 짧게 핵심만 적어 두면 나중에 목록에서 찾을 때 빠릅니다.

        본문 작성

        내용란에는 자유 형식으로 긴 글을 넣을 수 있습니다. 날짜별로 나누어 저장해 두면 특정 날짜에 무슨 생각을 했는지 나중에 추적하기 쉽습니다.

        색상·이모지

        제목 옆 색상·이모지를 사용하면 메모 종류(업무·아이디어·일기 등)를 구분할 수 있어 목록 스캔이 수월해집니다.

        위젯 창 크기 조절과 닫기

        창 크기

        위젯으로 연 창은 브라우저 창처럼 모서리를 드래그해 크기를 조절할 수 있습니다. 필요한 만큼만 띄워 두고 작업 공간을 확보할 수 있습니다.

        닫기

        창 닫기 버튼을 누르면 위젯만 닫히고 메인 탭의 데이터는 그대로 유지됩니다. 저장은 자동으로 되어 있으므로 별도 저장 버튼은 없습니다.

        여러 위젯

        달력·메모·타이머 등을 각각 위젯으로 띄워 두고 화면에 배치해 두면, 한 화면에서 여러 정보를 동시에 참고할 수 있습니다.

        저장 데이터 초기화 시 주의사항

        초기화가 되는 경우

        브라우저에서 사이트 데이터·캐시 삭제를 하거나, 시크릿 모드만 사용하면 해당 환경에 저장된 일정·루틴·메모는 사라집니다. 서버에 보관되지 않으므로 복구할 수 없습니다.

        기기 변경 시

        PC에서 쓰다가 다른 PC나 휴대폰에서 접속하면 그 환경에는 데이터가 없습니다. 각 기기·브라우저별로 데이터가 따로 쌓이므로 중요한 내용은 따로 백업해 두는 것이 좋습니다.

        의도적 삭제

        데이터를 지우고 싶다면 브라우저 설정에서 해당 사이트의 저장 데이터만 삭제하면 됩니다. 그러면 해당 브라우저에서의 모든 Jay Calendar 데이터가 제거됩니다.

        PC와 모바일에서 같은 계정 없이 사용하는 방법

        계정 불필요

        Jay Calendar는 로그인·회원가입이 없이 사용합니다. 브라우저만 있으면 접속해서 바로 달력·루틴·메모를 쓸 수 있습니다.

        기기별 데이터

        PC 브라우저에서 쓴 데이터는 그 PC에만 있고, 휴대폰 브라우저에서 쓴 데이터는 그 휴대폰에만 있습니다. 같은 계정으로 자동 동기화되는 구조가 아니므로, 필요한 경우 각 기기에서 따로 사용하면 됩니다.

        동일 기기 여러 탭

        같은 기기·같은 브라우저에서 여러 탭으로 열면 같은 저장소를 쓰므로, 한 탭에서 수정한 내용이 다른 탭에서도 반영될 수 있습니다.

        이달의 목표 문구 수정·삭제 방법

        수정

        달력 상단의 이달의 목표 입력 칸을 클릭하면 기존 문구가 선택됩니다. 그대로 새 문구를 입력하면 이전 내용이 대체되고, Enter를 누르면 저장됩니다.

        스타일 메뉴

        목표 칸 옆의 메뉴(⋮) 버튼을 누르면 이모지 변경 등 스타일 옵션이 나올 수 있습니다. 여기서 문구를 수정하거나 초기화할 수도 있습니다.

        삭제

        목표 문구를 모두 지우고 빈 칸으로 두면 해당 월의 목표는 비워진 상태로 저장됩니다. 다시 입력하면 새 목표가 저장됩니다.
