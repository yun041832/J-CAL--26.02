# 메타인지 훈련: 잘못된 학습을 알아채는 법

안녕하세요, 학습 효과와 뇌 과학을 설계하는 **제이캘린더(J-Calendar)**입니다.

**메타인지**는 '생각에 대한 생각'입니다. "내가 이 내용을 정말 이해했는가?"를 스스로 점검하는 능력이며, 이 훈련이 없으면 '알고 있다고 착각하는 상태'에 머물 수 있습니다. 제이캘린더가 메타인지 훈련 3단계를 정리합니다.

---

## 1. 잘못된 학습의 3가지 징후

| 징후 | 설명 |
| :--- | :--- |
| **재인출 실패** | 복습 없이 넘어가서 며칠 뒤 기억이 사라짐 |
| **표면 이해** | 핵심을 놓치고 세부만 암기 |
| **검증 생략** | 스스로 설명·퀴즈를 하지 않고 학습 종료 |

---

## 2. 메타인지 훈련 3단계

| 단계 | 질문 | 실천 |
| :--- | :--- | :--- |
| **① 예측** | "이걸 얼마나 빨리 배울 수 있을까?" | 학습 전 난이도·소요 시간 추정 |
| **② 점검** | "정말 이해했는가?" | 한 단락 후 자신에게 설명 |
| **③ 조정** | "어디가 부족한가?" | 설명이 막히는 부분 복습 |

제이캘린더에 '메타인지 체크' 태스크를 학습 세션 끝에 5분 추가하세요.

---

## 3. 제이캘린더의 수익형 인사이트

메타인지는 학습 효율을 2배로 끌어올리는 레버입니다. 제이캘린더로 학습 세션과 메타인지 체크 루틴을 기록하면 잘못된 학습 패턴을 조기에 발견할 수 있습니다.
