# 슬럼프 극복: 동기부여의 뇌신경학

안녕하세요, 동기와 수익성을 설계하는 **제이캘린더(J-Calendar)**입니다.

**슬럼프**는 단순한 게으름이 아니라 도파민·세로토닌 시스템의 일시적 둔화입니다. '하고 싶은데 몸이 안 움직인다'는 상태는 뇌의 보상 회로가 피로해졌다는 신호입니다. 제이캘린더가 뇌신경학 기반 슬럼프 극복법을 정리합니다.

---

## 1. 슬럼프의 뇌과학

도파민은 '예상 보상'에 반응합니다. 목표가 너무 크거나 보상이 먼거나 불명확하면 도파민 분비가 둔해집니다. 반대로 작은 성취·즉각 보상·명확한 마일스톤은 도파민을 자극해 동기를 복구합니다.

---

## 2. 슬럼프 극복 4단계

| 단계 | 실천 |
| :--- | :--- |
| **① 축소** | 목표를 1/10로 줄이기 (예: 1시간 공부 → 10분) |
| **② 트리거** | 기존 루틴에 붙이기 (예: 커피 마신 직후 10분) |
| **③ 보상** | 완료 즉시 작은 보상 (체크박스·기록) |
| **④ 기록** | 연속 일수 시각화 |

제이캘린더에 '슬럼프 극복' 태스크를 등록하고, 10분 단위로 실행 가능한 항목을 넣어 두세요.

---

## 3. 제이캘린더의 수익형 인사이트

슬럼프는 2~4주 지속될 수 있으나, 작은 성취의 연속으로 뇌의 보상 회로를 다시 활성화할 수 있습니다. 제이캘린더로 10분 루틴과 연속 기록을 관리하면 동기 복구에 도움이 됩니다.
